# ✈️ UAV Navigation System — Teknofest 2026
# ✈️ İHA Navigasyon Sistemi — Teknofest 2026

**Developers / Geliştiriciler:** Yağız Günay / Efe Çelik

A real-time 3D UAV (Unmanned Aerial Vehicle) route planning and simulation system.
Uses the **A\*** algorithm for obstacle detection, physics-based flight simulation, and interactive 3D visualization.

Gerçek zamanlı 3D İHA (İnsansız Hava Aracı) rota planlama ve simülasyon sistemi.
**A\*** algoritması ile engel tespiti, fizik tabanlı uçuş simülasyonu ve interaktif 3D görselleştirme sunar.

---

## 🚀 Quick Start / Hızlı Başlangıç

**Double-click `run.bat`. No additional setup required.**
**`run.bat` dosyasına çift tıklayın. Başka bir şey yapmanıza gerek yok.**

The system automatically / Sistem otomatik olarak:

1. Checks for Node.js — installs from `installers\` if missing / Node.js varlığını kontrol eder, yoksa `installers\` klasöründen kurar
2. Checks for Python — installs from `installers\` if missing / Python varlığını kontrol eder, yoksa `installers\` klasöründen kurar
3. Runs `npm install express` / `npm install express` ile Node bağımlılığını indirir
4. Runs `pip install numpy matplotlib scikit-learn` / Python kütüphanelerini indirir
5. Starts `server.js` / `server.js`'i başlatır
6. Opens `http://localhost:3000` in your browser / Tarayıcıda `http://localhost:3000` adresini açar

> ⚠️ **First run / İlk çalıştırmada:** Downloads may take a few minutes. / İndirmeler nedeniyle birkaç dakika sürebilir.
> If **"KAPATIP TEKRAR BAŞLAT"** appears on screen, close the window and run `run.bat` again. / Ekranda bu mesaj çıkarsa pencereyi kapatıp `run.bat`'ı tekrar çalıştırın.

## 🗺️ Usage Steps / Kullanım Adımları

### 1 — Coordinate Input / Koordinat Girişi

Enter **Start** and **Target** points as X / Y / Z in the left panel.
Sol panelde **Başlangıç** ve **Hedef** noktalarını X / Y / Z olarak girin.

- Valid range / Geçerli aralık: **0 – 50** (50×50×50 unit 3D space / birimlik 3D uzay)
- Default / Varsayılan: Start / Başlangıç `(5, 5, 5)` → Target / Hedef `(45, 45, 45)`

### 2 — Select Route Mode / Rota Modu Seçin

| Mode / Mod | What it does / Ne yapar | Safety / Güvenlik | Speed / Hız |
|------------|-------------------------|-------------------|-------------|
| **01 · Shortest / En Kısa** | No obstacle margin, direct path / Engel marjı yok, direkt yol | ⚠️ Risky / Riskli | ⚡ Maximum / Maksimum |
| **02 · Safest / En Güvenli** | Flies 3.5+ units from obstacles / Engellerden 3.5+ birim uzakta uçar | ✅ High / Yüksek | 🐢 Slow / Yavaş |
| **03 · Balanced / Dengeli** | Balances safety and speed / Güvenlik ve hız dengesi | ⚖️ Medium / Orta | 🚀 Optimal |

### 3 — Start Simulation / Simülasyonu Başlat

Click **▶ SİMÜLASYONU BAŞLAT**. Follow the 5-step progress bar:
**▶ SİMÜLASYONU BAŞLAT** butonuna tıklayın. 5 adımlı ilerleme çubuğunu takip edin:

```
① Environment & Obstacles Setup   / Ortam & Engeller Kuruluyor
② A* Algorithm Computing Route    / A* Algoritması Rota Hesaplıyor
③ Physics Simulation Running      / Fizik Simülasyonu Çalışıyor
④ Animation Rendering             / Animasyon Render Ediliyor
⑤ Performance Report Preparing    / Performans Raporu Hazırlanıyor
```

When complete / Tamamlandığında:
- A **GIF animation** and **PNG report** appear on the right side of the page / Sayfanın sağında **GIF animasyonu** ve **PNG raporu** belirir
- Separate **Download** links appear for each / Her ikisi için ayrı **İndir** bağlantısı çıkar

### 4 — 3D GPU Sandbox

Click **⬡ GPU RENDER BAŞLAT** / **⬡ GPU RENDER BAŞLAT** butonuna tıklayın.

| Object / Nesne | Meaning / Anlamı |
|----------------|-----------------|
| 🔵 Blue drone / Mavi drone | Safe / slow route / Güvenli / yavaş rota |
| 🔴 Red drone / Kırmızı drone | Fast / direct route / Hızlı / direkt rota |
| Colored spheres / Renkli küreler | Fixed obstacles / Sabit engeller |
| Blue line / Mavi çizgi | Blue drone's trail / Mavi dronun geçtiği iz |
| Red line / Kırmızı çizgi | Red drone's trail / Kırmızı dronun geçtiği iz |

> Can be opened without running a simulation — coordinates are read automatically from the panel.
> Simülasyon çalıştırılmadan da açılabilir — koordinatlar panelden otomatik okunur.

## 🧠 Technical Details / Teknik Detaylar

### A\* Engine / A\* Motoru — `app.py`

Two path-finders run with different parameters per mode.
İki path-finder çalışır, mod'a göre farklı parametreler alır:

| Parameter / Parametre | Mode 01 / Mod 01 | Mode 02 / Mod 02 | Mode 03 / Mod 03 |
|-----------------------|-----------------|-----------------|-----------------|
| Safe safetyMargin | 0.1 | 3.5 | 2.0 |
| Safe safetyFactor | 0.0 | 25.0 | 12.0 |
| Fast safetyMargin | 0.1 | 2.0 | 0.8 |
| Fast safetyFactor | 0.0 | 10.0 | 2.0 |

**safetyMargin:** Minimum distance to keep from obstacle surface (units). / Engel yüzeyinden tutulması gereken minimum mesafe (birim).

**safetyFactor:** Cost penalty for passing through hazardous zones — higher values make A\* actively avoid these areas. / Tehlikeli bölgelerden geçişin maliyet cezası — yüksekse A\* bu alanlardan aktif olarak kaçar.

**Path Smoothing:** After path-finding, a line-of-sight test removes unnecessary waypoints for a more natural flight curve. / Yol bulunduktan sonra görüş hattı testi yapılır; gereksiz kırılma noktaları çıkarılarak dron daha doğal bir eğride uçar.

### Physics Simulation / Fizik Simülasyonu — `Drone3D`

- Force-based movement / Kuvvet bazlı hareket: `F_steering = v_desired − v_current`
- Acceleration limit for realistic speed-up/slow-down / İvme sınırı (gerçekçi hızlanma/yavaşlama)
- Dynamic braking within 2 units of target / Hedefe 2 birim kala dinamik frenleme
- Energy calculation / Enerji hesabı: `E = 0.5 × m × v² × Δt`
- Crash detection: obstacle radius + drone radius < distance, checked every step / Kaza tespiti: her adımda kontrol edilir

### Fixed Obstacles / Sabit Engeller

| Center / Merkez | Radius / Yarıçap |
|-----------------|-----------------|
| (15, 15, 15) | 4 |
| (25, 20, 25) | 5 |
| (35, 30, 20) | 4.5 |
| (20, 40, 30) | 4 |
| (30, 25, 40) | 5 |

### Server / Sunucu — `server.js`

- Express, port **3000**
- `POST /calculate` → spawns `app.py` as subprocess / `app.py`'yi subprocess olarak başlatır
- Results stream to browser via **streaming NDJSON** (drives progress bar) / Sonuçlar streaming NDJSON ile tarayıcıya akar
- `public/` folder served as static files / statik dosya olarak sunulur

## 🔧 Troubleshooting / Sorun Giderme

**Browser didn't open / Tarayıcı açılmadı:**
Manually navigate to `http://localhost:3000`. / Manuel olarak `http://localhost:3000` adresini açın.

**"Node.js not found" / "Node.js bulunamadı" hatası:**
Close `run.bat`, ensure `installers\node-setup.msi` exists, then run `run.bat` again. / `run.bat`'ı kapatın, MSI dosyasının mevcut olduğundan emin olun ve tekrar çalıştırın.

**"Python not found" / "Python bulunamadı" hatası:**
Install via `installers\python-setup.exe`. During installation, check **"Add Python to PATH"**. Then run `run.bat` again. / Kurulum sırasında **"Add Python to PATH"** seçeneğini işaretleyin. Ardından `run.bat`'ı tekrar çalıştırın.

**3D sandbox black / blank — 3D sandbox siyah / boş görünüyor:**
Use Chrome or Edge. Update GPU drivers. Click **⬡ GPU RENDER BAŞLAT** again. / Chrome veya Edge kullanın. GPU sürücülerinizi güncelleyin.

**Route not found / Rota bulunamadı:**
One coordinate may be inside an obstacle. Try different coordinates or switch to Mode 02. / Koordinatlardan biri engelin içinde kalıyor olabilir. Farklı koordinat deneyin veya Mod 02'ye geçin.

**`pip` command not recognized / `pip` komutu tanınmıyor:**
Python was installed without PATH selection. Reinstall via `python-setup.exe` and check **"Add Python to PATH"**. / Python kurulumunda PATH seçimi yapılmamıştır. Yeniden kurun ve kutuyu işaretleyin.

## 📊 How to Read the PNG Report / PNG Raporu Nasıl Okunur

| Chart / Grafik | What it shows / Ne gösterir |
|----------------|------------------------------|
| **Duration (s) / Süre (s)** | Total time of A\* + physics simulation in Python / A\* + fizik simülasyonunun toplam süresi |
| **Energy (J) / Enerji (J)** | Flight energy calculated via `E = ½mv²Δt` / Uçuş enerjisi |

Blue bar = safe route / Mavi çubuk = güvenli rota, Red bar = fast route / Kırmızı çubuk = hızlı rota.

The safe route travels steadily and slowly, typically consuming less energy; the fast route arrives much sooner but uses more energy.
Güvenli rota sabit ve yavaş ilerlediği için genellikle daha az enerji harcar; hızlı rota çok daha erken varır ama daha fazla enerji tüketir.
